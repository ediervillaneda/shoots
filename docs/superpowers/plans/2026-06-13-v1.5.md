# v1.5 Implementation Plan -- Starfall

## Antes de empezar

- [ ] Leer el design spec completo: docs/superpowers/specs/2026-06-13-v1.5-design.md
- [ ] Confirmar que todos los tests anteriores pasan: python -m pytest tests/ -v
- [ ] Crear rama: git checkout develop && git checkout -b v1.5/nuevos-powerups

## Tasks -- seguir orden del design spec

### Task 1: Constantes y settings

- [ ] Crear o modificar modulos en src/settings/ segun spec
- [ ] Verificar imports: python -c "from src.settings import *; print('OK')"

### Task 2: Entidades nuevas o modificadas

- [ ] Implementar archivos en src/entities/ segun spec
- [ ] Verificar import de cada entidad nueva sin crash
- [ ] Checkpoint: python -m pytest tests/ -v --tb=short

### Task 3: Sistemas nuevos o modificados

- [ ] Implementar archivos en src/systems/ segun spec
- [ ] Checkpoint: python -m pytest tests/ -v --tb=short

### Task 4: Integracion en GameplayScene / Game

- [ ] Aplicar todos los cambios en src/scenes/ y src/game.py segun spec
- [ ] Checkpoint: python -m pytest tests/ -v --tb=short

### Task 5: Tests nuevos

- [ ] Crear o modificar archivos de test segun spec
- [ ] Suite completa: python -m pytest tests/ -v

### Task 6: Verificacion manual

- [ ] python main.py -- juego arranca sin crash
- [ ] Verificar cada criterio de completitud del spec manualmente
- [ ] Sin crashes en 3 minutos de juego continuo
