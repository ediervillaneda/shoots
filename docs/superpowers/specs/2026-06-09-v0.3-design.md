## Componentes

### `Player` — `src/entities/player.py`

**Atributos nuevos:**
- `self.lives = PLAYER_LIVES`
- `self.invincible = False`
- `self.invincible_until = 0`

**Método nuevo `take_damage(now)`:**
```python
def take_damage(self, now):
    if self.invincible:
        return
    self.lives -= 1
    self.invincible = True
    self.invincible_until = now + INVINCIBILITY_MS
```

**`update(dt)` — cambios:**
```python
now = pygame.time.get_ticks()
if self.invincible and now >= self.invincible_until:
    self.invincible = False
    self.image.set_alpha(255)

if self.invincible:
    visible = (now // 100) % 2 == 0
    self.image.set_alpha(255 if visible else 0)
```

### `GameplayScene` — `src/scenes/gameplay.py`

**Atributos nuevos:**
- `self.score = 0`
- `self.state = "start"`   # "start" | "playing" | "game_over"
- `self._font = pygame.font.SysFont(None, HUD_FONT_SIZE)`

**Método `_reset()`:**
```python
def _reset(self):
    self.player = Player()
    self.bullets = pygame.sprite.Group()
    self.enemies = pygame.sprite.Group()
    self.all_sprites = pygame.sprite.Group(self.player)
    self.last_spawn = pygame.time.get_ticks()
    self.score = 0
    self.state = "playing"
```

**`process_input(events)` — cambios:**
```python
if self.state in ("start", "game_over"):
    if keys[pygame.K_SPACE]:
        self._reset()
    return                  # no más input mientras no se juega

# resto del input (WASD + disparo) solo en "playing"
```

**`update(dt)` — cambios:**
```python
if self.state != "playing":
    return

# ... lógica existente ...

# colisión player↔enemies (nueva)
now = pygame.time.get_ticks()
hit_enemies = pygame.sprite.spritecollide(self.player, self.enemies, True)
for enemy in hit_enemies:
    self.score += enemy.points
    self.player.take_damage(now)

# bullets↔enemies: agregar score cuando enemy muere
hits = pygame.sprite.groupcollide(self.bullets, self.enemies, True, False)
for enemies_hit in hits.values():
    for enemy in enemies_hit:
        if enemy.alive():
            enemy.take_damage(BULLET_DAMAGE)
            if not enemy.alive():
                self.score += enemy.points

if self.player.lives <= 0:
    self.state = "game_over"
```

**`render(screen)` — cambios:**
```python
screen.fill((0, 0, 0))
self.all_sprites.draw(screen)

# HUD
score_surf = self._font.render(f"SCORE: {self.score}", True, HUD_COLOR)
screen.blit(score_surf, (HUD_MARGIN, HUD_MARGIN))

lives_text = "♥ " * self.player.lives
lives_surf = self._font.render(lives_text.strip(), True, (220, 50, 50))
screen.blit(lives_surf, (SCREEN_W - lives_surf.get_width() - HUD_MARGIN, HUD_MARGIN))

# Overlays de estado
if self.state == "start":
    self._draw_overlay(screen, "STARFALL", "PRESS SPACE TO PLAY")
elif self.state == "game_over":
    self._draw_overlay(screen, "GAME OVER", "PRESS SPACE TO RESTART")
```

**Método `_draw_overlay(screen, title, subtitle)`:**
```python
def _draw_overlay(self, screen, title, subtitle):
    overlay = pygame.Surface((SCREEN_W, SCREEN_H), pygame.SRCALPHA)
    overlay.fill((0, 0, 0, 160))
    screen.blit(overlay, (0, 0))
    title_surf = self._font.render(title, True, (255, 255, 255))
    sub_surf = self._font.render(subtitle, True, (180, 180, 180))
    cx = SCREEN_W // 2
    cy = SCREEN_H // 2
    screen.blit(title_surf, (cx - title_surf.get_width() // 2, cy - 30))
    screen.blit(sub_surf, (cx - sub_surf.get_width() // 2, cy + 10))
```

## Flujo completo

```
GameplayScene.update(dt):
  if state != "playing": return

  player.update(dt)          → mueve, gestiona invencibilidad/parpadeo
  bullets.update(dt)
  enemies.update(dt)
  spawn_check()              → timer simple, sin escalado
  bullets↔enemies            → bullet muere, enemy.take_damage(); si muere: score += points
  player↔enemies             → enemy kill + player.take_damage(now) + score += points
  if player.lives ≤ 0        → state = "game_over"

GameplayScene.process_input(events):
  if state in ("start","game_over"):
    SPACE → _reset() → state = "playing"
    return
  WASD → velocidad player
  SPACE → player.shoot()

GameplayScene.render(screen):
  all_sprites.draw()
  HUD: score top-left, vidas top-right
  if start/game_over: overlay semitransparente con texto
```

## Criterio de completitud v0.3

- Pantalla inicial "PRESS SPACE TO PLAY" — juego no empieza hasta presionar espacio
- Matar Scout: +100 pts; matar Fighter: +250 pts (bala o colisión)
- Score visible top-left en tiempo real
- Player pierde vida al tocar enemy; enemy desaparece
- Player parpadea 2s tras daño, invulnerable ese tiempo
- 5 vidas mostradas top-right como corazones rojos
- Con 0 vidas: overlay "GAME OVER / PRESS SPACE TO RESTART"
- SPACE en game over reinicia partida completa (score=0, lives=5, sin enemies)
- Sin crashes en 5 minutos de juego continuo
