## Estructura de archivos

```
src/entities/
  enemy.py        ← Enemy(Sprite) base: hp, points, take_damage(), update(dt)
  scout.py        ← Scout(Enemy): movimiento recto hacia abajo
  fighter.py      ← Fighter(Enemy): zigzag horizontal con sin()
src/settings.py   ← agregar constantes Scout, Fighter, spawning
src/scenes/
  gameplay.py     ← modificar: grupo enemies, spawn timer, groupcollide
```

Sin `src/systems/spawning.py` — spawn simple en `GameplayScene`. SpawnSystem con oleadas va en v0.3.

## Componentes

### `Enemy(Sprite)` — `src/entities/enemy.py`

Clase base para todos los tipos de enemigo.

- Atributos: `hp`, `points`, `x` (float), `y` (float)
- `take_damage(amount)`: reduce hp; llama `kill()` si hp ≤ 0
- `update(dt)`: mueve hacia abajo; llama `kill()` si `rect.top > SCREEN_H`
- Subclases definen `image`, `rect`, `hp`, `points` y sobreescriben `update(dt)`

### `Scout(Enemy)` — `src/entities/scout.py`

- Rect placeholder 40×40, color `SCOUT_COLOR`
- Spawn: `rect.centerx = x`, `rect.bottom = 0` (entra desde arriba)
- Movimiento: `y += SCOUT_SPEED * dt` (recto hacia abajo)
- hp=`SCOUT_HP`=1, points=`SCOUT_POINTS`=100

### `Fighter(Enemy)` — `src/entities/fighter.py`

- Rect placeholder 44×44, color `FIGHTER_COLOR`
- Spawn: igual que Scout
- Movimiento: baja con `FIGHTER_SPEED`; X oscila con:
  ```python
  self.elapsed += dt
  offset = math.sin(self.elapsed * FIGHTER_FREQUENCY * 2 * math.pi) * FIGHTER_AMPLITUDE
  self.rect.x = int(self.origin_x + offset)
  ```
  `origin_x` = X de spawn (centro del zigzag)
- hp=`FIGHTER_HP`=3, points=`FIGHTER_POINTS`=250

## Modificaciones a `GameplayScene`

### Nuevos atributos en `__init__`:
```python
self.enemies = pygame.sprite.Group()
self.last_spawn = pygame.time.get_ticks()
```
`enemies` también se agrega a `all_sprites` al hacer spawn.

### `spawn_enemy()` — método privado:
- Elige `Scout` o `Fighter` al azar (50/50 con `random.choice`)
- `x = random.randint(margin, SCREEN_W - margin)` donde `margin = max(SCOUT_W, FIGHTER_W) // 2`
- Agrega a `self.enemies` y `self.all_sprites`

### `update(dt)` — cambios:
```python
self.enemies.update(dt)

now = pygame.time.get_ticks()
if now - self.last_spawn >= SPAWN_INTERVAL:
    self.last_spawn = now
    self.spawn_enemy()

hits = pygame.sprite.groupcollide(self.bullets, self.enemies, True, False)
for enemy in hits:
    enemy.take_damage(1)
```

### `render(screen)` — sin cambios
`all_sprites.draw(screen)` dibuja enemies automáticamente por estar en el grupo.

## Flujo completo

```
GameplayScene.update(dt):
  player.update(dt)
  bullets.update(dt)
  enemies.update(dt)
  spawn_check(now)         → Scout o Fighter aleatorio desde arriba
  groupcollide()           → bullet muere, enemy.take_damage(1)
                           → enemy.kill() si hp ≤ 0

GameplayScene.render(screen):
  screen.fill((0,0,0))
  all_sprites.draw(screen) → player + bullets + enemies
```

## Criterio de completitud v0.2

- Scouts aparecen desde arriba y bajan en línea recta
- Fighters aparecen y zigzaguean horizontalmente
- Balas del jugador matan Scout en 1 hit, Fighter en 3 hits
- Enemigos desaparecen al salir por el borde inferior
- Player no recibe daño (invulnerable en v0.2)
- Sin crashes en 5 minutos de juego con spawns continuos
