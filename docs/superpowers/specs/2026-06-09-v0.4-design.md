## Componentes

### `EnemyBullet` — `src/entities/enemy_bullet.py`

Bala disparada por enemies. Se mueve hacia abajo.

```python
import pygame
from src.settings import (
    ENEMY_BULLET_SPEED, ENEMY_BULLET_COLOR,
    ENEMY_BULLET_W, ENEMY_BULLET_H, SCREEN_H,
)


class EnemyBullet(pygame.sprite.Sprite):
    def __init__(self, x, y):
        super().__init__()
        self.image = pygame.Surface((ENEMY_BULLET_W, ENEMY_BULLET_H))
        self.image.fill(ENEMY_BULLET_COLOR)
        self.rect = self.image.get_rect(centerx=x, top=y)
        self.y = float(self.rect.y)

    def update(self, dt):
        self.y += ENEMY_BULLET_SPEED * dt
        self.rect.y = int(self.y)
        if self.rect.top > SCREEN_H:
            self.kill()
```

### `SpawnSystem` — `src/systems/spawning.py`

Gestiona oleadas y spawn de enemies. No tiene referencias a pygame.sprite directo — retorna objetos Enemy listos para que `GameplayScene` los agregue a sus grupos.

```python
import random
import pygame
from src.entities.scout import Scout
from src.entities.fighter import Fighter
from src.settings import (
    SCREEN_W, SCOUT_W, FIGHTER_W,
    WAVE_TIME_THRESHOLD, WAVE_KILL_THRESHOLD,
    WAVE_SPEED_FACTOR, WAVE_SPAWN_MIN, WAVE_SPAWN_MIN_CAP,
    WAVE_FIGHTER_BASE, WAVE_FIGHTER_INC, WAVE_FIGHTER_CAP,
    SCOUT_SPEED, FIGHTER_SPEED,
)


class SpawnSystem:
    def __init__(self):
        self.wave = 0
        self.kills = 0
        self._last_spawn = pygame.time.get_ticks()
        self._wave_start = pygame.time.get_ticks()

    def register_kill(self):
        self.kills += 1
        if self.wave >= 3 and self.kills % WAVE_KILL_THRESHOLD == 0:
            self.wave += 1

    def update(self, now):
        """Retorna lista de Enemy nuevos a agregar, o [] si no toca spawnear."""
        if self.wave < 3:
            if now - self._wave_start >= WAVE_TIME_THRESHOLD:
                self.wave += 1
                self._wave_start = now

        if now - self._last_spawn >= self._spawn_interval():
            self._last_spawn = now
            return [self._make_enemy()]
        return []

    def _spawn_interval(self):
        return max(WAVE_SPAWN_MIN_CAP, WAVE_SPAWN_MIN - self.wave * 150)

    def _fighter_prob(self):
        return min(WAVE_FIGHTER_CAP, WAVE_FIGHTER_BASE + self.wave * WAVE_FIGHTER_INC)

    def _make_enemy(self):
        margin = max(SCOUT_W, FIGHTER_W) // 2
        x = random.randint(margin, SCREEN_W - margin)
        speed_bonus = self.wave * WAVE_SPEED_FACTOR
        if random.randint(1, 100) <= self._fighter_prob():
            f = Fighter(x)
            f.speed_bonus = speed_bonus   # usado en Fighter.update() → ver nota
            return f
        s = Scout(x)
        s.speed_bonus = speed_bonus
        return s
```

**Nota sobre speed_bonus:** En v0.4, Scout y Fighter leen `self.speed_bonus` en `update(dt)` si existe, para aplicar velocidad extra. Cada subclase agrega:
```python
bonus = getattr(self, "speed_bonus", 0.0)
self.y += (SCOUT_SPEED + bonus) * dt   # Scout
self.y += (FIGHTER_SPEED + bonus) * dt  # Fighter
```

Esto evita cambiar la firma de `__init__` ni romper tests existentes.

### `GameplayScene` — modificaciones

**Atributos nuevos en `__init__` y `_reset()`:**
```python
self.enemy_bullets = pygame.sprite.Group()
self.spawn_system = SpawnSystem()
```

`_reset()` completo con los nuevos atributos:
```python
def _reset(self):
    self.player = Player()
    self.bullets = pygame.sprite.Group()
    self.enemies = pygame.sprite.Group()
    self.enemy_bullets = pygame.sprite.Group()
    self.all_sprites = pygame.sprite.Group(self.player)
    self.last_spawn = pygame.time.get_ticks()   # eliminado en v0.4 — lo gestiona SpawnSystem
    self.score = 0
    self.spawn_system = SpawnSystem()
    self.state = "playing"
```

Nota: `self.last_spawn` en `GameplayScene` ya no se usa — `SpawnSystem` gestiona su propio timer. Se elimina en v0.4.

**`update(dt)` — cambios:**

Reemplazar bloque de spawn timer existente y agregar lógica de enemy bullets:

```python
now = pygame.time.get_ticks()

# Spawn via SpawnSystem
for enemy in self.spawn_system.update(now):
    self.enemies.add(enemy)
    self.all_sprites.add(enemy)

# Fighter dispara
for sprite in self.enemies:
    if isinstance(sprite, Fighter):
        eb = sprite.shoot(now)
        if eb:
            self.enemy_bullets.add(eb)
            self.all_sprites.add(eb)

# Enemy bullets mueven
self.enemy_bullets.update(dt)

# Colisión enemy_bullets ↔ player
eb_hits = pygame.sprite.spritecollide(self.player, self.enemy_bullets, True)
for _ in eb_hits:
    self.player.take_damage(now)

# Colisión player ↔ enemies (existente — agregar register_kill)
hit_enemies = pygame.sprite.spritecollide(self.player, self.enemies, True)
for enemy in hit_enemies:
    self.score += enemy.points
    self.player.take_damage(now)
    self.spawn_system.register_kill()

# Colisión bullets ↔ enemies (existente — agregar register_kill)
hits = pygame.sprite.groupcollide(self.bullets, self.enemies, True, False)
for enemies_hit in hits.values():
    for enemy in enemies_hit:
        if enemy.alive():
            enemy.take_damage(BULLET_DAMAGE)
            if not enemy.alive():
                self.score += enemy.points
                self.spawn_system.register_kill()
```

## Flujo completo

```
GameplayScene.update(dt):
  player.update(dt)
  bullets.update(dt)
  enemies.update(dt)
  enemy_bullets.update(dt)

  now = pygame.time.get_ticks()
  SpawnSystem.update(now)         → nuevo enemy si toca; avanza wave
  Fighter.shoot(now)              → EnemyBullet si cooldown ok
  enemy_bullets ↔ player          → kill bullet + player.take_damage(now)
  player ↔ enemies                → score + take_damage + register_kill
  bullets ↔ enemies               → take_damage; si muere: score + register_kill
  lives ≤ 0                       → state = "game_over"

GameplayScene.render(screen):
  all_sprites.draw()
  HUD: SCORE top-left | WAVE top-center | ♥ top-right
```

## Criterio de completitud v0.4

- Fighter vivo dispara una bala roja recta hacia abajo cada 2s
- Bala enemiga daña al player (invencibilidad aplica, bala desaparece)
- Oleadas 0-2 suben cada 30s; oleada 3+ sube cada 10 kills
- Cada oleada: enemies más rápidos (`wave * 0.2` px/s extra), spawn más frecuente, más Fighters
- `WAVE: N` visible en HUD top-center
- Sin crashes en 5 minutos de juego continuo
