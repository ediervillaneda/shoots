# v2.0 Design -- Starfall (Release Final)
**Fecha:** 2026-06-13 | **Version:** v2.0
**Scope:** Boss fase 3, slow motion, disparo auto, balance, build
**Base:** v1.9

## Boss fase 3
Umbral: boss.hp <= BOSS_HP * 0.25 (25%).
BOSS_PHASE3_HP_THRESHOLD=0.25, BOSS_SHOOT_INTERVAL_P3=600
BOSS_SPIRAL_ANGLE_INC=15.0, BOSS_ERRATIC_SPEED=280, BOSS_ERRATIC_CHANGE_MS=400

Movimiento erratico: cambia direccion horizontal cada 400ms a 280 px/s.

Espiral: 2 balas por disparo (angle y angle+180). _spiral_angle += 15 cada disparo.

phase property: ratio > 0.5 -> 1; ratio > 0.25 -> 2; else -> 3.

## Slow motion al matar Boss
SLOWMO_DURATION_MS=800, SLOWMO_SCALE=0.25.
Al confirmar muerte del boss: _slowmo_ms = 800.
En update(dt): si _slowmo_ms > 0, dt *= 0.25.

## Disparo automatico
Tecla A alterna _autofire. Si True: dispara aunque Space no este presionado.

## Balance final
BULLET_DAMAGE_DEFAULT: Interceptor muere en 8-10 balas.
BOSS_HP: Boss dura 30-60s en wave 10.
PLASMA_DAMAGE: Mata Scout de 1 disparo.
LASER_DAMAGE_PER_S: Mata Scout en ~0.5s.

## Build
Starfall.spec: assets/ completo, console=False.
CI: trigger en tags v2.*, tests, pyinstaller, upload artifact.
README y CHANGELOG actualizados.

- Boss fase 3 activa a 25% HP: espiral y movimiento erratico
- Slow motion 800ms al matar boss
- Tecla A alterna disparo automatico
