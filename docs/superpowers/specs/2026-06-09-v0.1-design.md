# v0.1 Design — Starfall

**Fecha:** 2026-06-09
**Versión:** v0.1
**Scope:** Ventana, nave jugador, movimiento WASD, disparo con Espacio
**Assets:** Placeholders (rectángulos de color, sin imágenes reales)

---

## Estructura de archivos

```
main.py                    ← instancia Game, llama game.run()
src/
  settings.py              ← todas las constantes
  game.py                  ← Game class, loop principal, gestiona scene activa
  scenes/
    __init__.py
    gameplay.py            ← GameplayScene: input → update → render
  entities/
    __init__.py
    player.py              ← Player(pygame.sprite.Sprite)
    bullet.py              ← Bullet(pygame.sprite.Sprite)
assets/                    ← vacío (placeholders en código)
```

`src/systems/` y `src/assets.py` no se crean en v0.1.

## Componentes

### `Game` (src/game.py)
- Inicializa pygame, crea ventana `SCREEN_W × SCREEN_H`, clock
- Instancia `GameplayScene` como scene activa (referencia directa, sin SceneManager)
- `run()`: event loop, ESC → quit, delega input/update/render a scene activa
- `dt` en segundos: `clock.tick(FPS) / 1000`

### `GameplayScene` (src/scenes/gameplay.py)
- Posee `Player` y `pygame.sprite.Group` para bullets
- `process_input(events)`: delega teclas a player, agrega bullets al grupo
- `update(dt)`: actualiza player y bullets, elimina bullets fuera de pantalla
- `render(screen)`: fill negro, dibuja player y bullets

### `Player` (src/entities/player.py)
- Hereda `pygame.sprite.Sprite`
- Placeholder: rectángulo 40×50, `PLAYER_COLOR`
- Posición inicial: centro horizontal, 80px desde el borde inferior
- Movimiento: vector WASD normalizado × `PLAYER_SPEED`, clamped a límites de pantalla
- `shoot(now) → Bullet | None`: retorna `Bullet` si cooldown elapsed, si no `None`
- Cooldown: `BULLET_COOLDOWN` ms, trackea `self.last_shot`

### `Bullet` (src/entities/bullet.py)
- Hereda `pygame.sprite.Sprite`
- Placeholder: rectángulo 6×14, `BULLET_COLOR`
- Spawn: centro-X del player, borde superior del player
- Movimiento: sube `BULLET_SPEED` px/s × dt
- `update(dt)`: mueve y llama `kill()` si `rect.bottom < 0`

## Game loop

```
Game.run():
  while running:
    events = pygame.event.get()
    ESC → running = False

    scene.process_input(events)
      Player: WASD → velocity vector normalizado
      Space held → player.shoot(now) → si retorna Bullet → bullets.add()

    scene.update(dt)
      player.update(dt) → pos += velocity * PLAYER_SPEED * dt, clamp
      bullets.update(dt) → pos -= BULLET_SPEED * dt, kill si fuera de pantalla

    scene.render(screen)
      screen.fill((0, 0, 0))
      screen.blit / draw player
      bullets.draw(screen)

    clock.tick(FPS)
```

ESC manejado en `Game.run()`, no en la scene.

## Criterio de completitud v0.1

- Ventana 480×640 abre a 60 FPS estables
- Nave (rectángulo) se mueve en 8 direcciones sin salir de pantalla
- Espacio dispara bullets que suben y desaparecen al salir de pantalla
- Cooldown de disparo funciona (no spam infinito)
- Sin crashes en 5 minutos de uso
