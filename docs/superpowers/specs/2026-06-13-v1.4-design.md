## Player.shoot() -- gun_upgrade con divergencia

Balas centrales a 0 grados. Balas laterales llevan angle_deg proporcional.

Nivel 2 (2 balas):
angle_deg = +/- GUN_UPGRADE_DIVERGE_DEG * 0.5

Nivel 3-4 (3 balas):
- izq: angle_deg = -GUN_UPGRADE_DIVERGE_DEG
- centro: 0
- der: +GUN_UPGRADE_DIVERGE_DEG

Nivel 5-6 (5 balas):
- -2x, -1x, 0, +1x, +2x de GUN_UPGRADE_DIVERGE_DEG

## Logica de tint en shoot()

`python
damage = BULLET_DAMAGE_DEFAULT
tint = None
if "rapid_fire" in self.active_powerups:
    damage = max(1, int(BULLET_DAMAGE_DEFAULT * 0.5))
    tint = BULLET_TINT_RAPIDFIRE
if "shield" in self.active_powerups:
    tint = BULLET_TINT_SHIELD
`

Precedencia: shield sobreescribe tint de rapid_fire, pero damage de rapid_fire se mantiene.

def test_gun_upgrade_lvl3_balas_divergen():
    p = Player()

    assert len(balas) == 3
    assert balas[0]._vx < 0
    assert balas[1]._vx == pytest.approx(0.0)

def test_shield_activo_tinta_balas():
    p = Player()
    p.active_powerups.add("shield")
    balas = p.shoot(0)
    assert balas[0].tint == BULLET_TINT_SHIELD

def test_rapidfire_activo_damage_reducido():
    p = Player()
    p.active_powerups.add("rapid_fire")
    balas = p.shoot(0)
    assert balas[0].damage < BULLET_DAMAGE_DEFAULT
`
