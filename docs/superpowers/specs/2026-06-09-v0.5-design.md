### `src/assets.py`

Carga lazy con cache. No carga nada al importar — primer `get()` carga y cachea.

```python
import pygame
from pathlib import Path

_BASE = Path(__file__).parent.parent / "assets"
_cache: dict[str, pygame.Surface] = {}

def get(path: str) -> pygame.Surface:
    if path not in _cache:
        _cache[path] = pygame.image.load(_BASE / path).convert_alpha()
    return _cache[path]
```

Llamado: `assets.get("ship/insect-1.png")`.

### `Kamikaze` — `src/entities/kamikaze.py`

Enemy que persigue al player cada frame. Usa sprite `insect-1.png` (48×48).

```python
import math
import pygame
import src.assets as assets
from src.entities.enemy import Enemy
from src.settings import KAMIKAZE_SPEED, KAMIKAZE_W, KAMIKAZE_H, KAMIKAZE_HP, KAMIKAZE_POINTS


class Kamikaze(Enemy):
    def __init__(self, x):
        super().__init__(x, KAMIKAZE_W, KAMIKAZE_H, (180, 50, 220), KAMIKAZE_HP, KAMIKAZE_POINTS)
        raw = assets.get("ship/insect-1.png")
        self.image = pygame.transform.scale(raw, (KAMIKAZE_W, KAMIKAZE_H))
        self.rect = self.image.get_rect(centerx=x, bottom=0)

    def update(self, dt):
        speed = KAMIKAZE_SPEED + getattr(self, "speed_bonus", 0.0)
        target = getattr(self, "target", None)
        if target:
            dx = target.rect.centerx - self.rect.centerx
            dy = target.rect.centery - self.rect.centery
            dist = max(1.0, math.hypot(dx, dy))
            self.x += (dx / dist) * speed * dt
            self.y += (dy / dist) * speed * dt
        else:
            self.y += speed * dt
        self.rect.x = int(self.x)
        self.rect.y = int(self.y)
        super().update(dt)
```

`target` se asigna externamente desde `GameplayScene` al spawnear:
```python
for enemy in self.spawn_system.update(now):
    if isinstance(enemy, Kamikaze):
        enemy.target = self.player
    self.enemies.add(enemy)
    self.all_sprites.add(enemy)
```

Si `target` no está aseteado (ej. tests), cae recto hacia abajo.

### `PowerUp` — `src/entities/powerup.py`

Un solo archivo, clase única. `kind` determina sprite y efecto.

```python
import pygame
import src.assets as assets
from src.settings import POWERUP_SPEED, POWERUP_W, POWERUP_H, SCREEN_H

POWERUP_ASSETS = {
    "rapid_fire":  "ship/bonus_time.png",
    "shield":      "ship/bonus_shield.png",
    "double_shot": "ship/laser-2.png",
    "triple_shot": "ship/laser-3.png",
    "extra_life":  "ship/bonus_life.png",
}

POWERUP_KINDS = list(POWERUP_ASSETS)


class PowerUp(pygame.sprite.Sprite):
    def __init__(self, kind: str, x: int, y: int):
        super().__init__()
        raw = assets.get(POWERUP_ASSETS[kind])
        self.image = pygame.transform.scale(raw, (POWERUP_W, POWERUP_H))
        self.rect = self.image.get_rect(centerx=x, top=y)
        self.y = float(self.rect.y)
        self.kind = kind

    def update(self, dt):
        self.y += POWERUP_SPEED * dt
        self.rect.y = int(self.y)
        if self.rect.top > SCREEN_H:
            self.kill()
```

```python
def take_damage(self, now):
    if self.invincible:
        return
    if "shield" in self.active_powerups:
        self.active_powerups.discard("shield")
        return                           # hit absorbido, sin perder vida
    self.lives = max(0, self.lives - 1)
    self.active_powerups.clear()         # todos los power-ups se pierden al recibir daño
    self.invincible = True
    self.invincible_until = now + INVINCIBILITY_MS

**`shoot(now)` — retorna `list[Bullet]`** (cambio de firma):
```python
def shoot(self, now):
    cooldown = BULLET_COOLDOWN
    if "rapid_fire" in self.active_powerups:
        cooldown = int(BULLET_COOLDOWN * RAPIDFIRE_COOLDOWN_MULT)
    if now - self.last_shot < cooldown:
        return []
    self.last_shot = now
    cx = self.rect.centerx
    top = self.rect.top
    if "triple_shot" in self.active_powerups:
        return [Bullet(cx, top), Bullet(cx - 16, top), Bullet(cx + 16, top)]
    if "double_shot" in self.active_powerups:
        return [Bullet(cx - 10, top), Bullet(cx + 10, top)]
    return [Bullet(cx, top)]
```

**Nota de precedencia:** `triple_shot` se chequea antes que `double_shot`. Si ambos están activos simultáneamente, `triple_shot` gana.

**`_make_enemy()` — modificado:**

Primero elige si es Kamikaze. Si no, usa lógica existente Scout/Fighter:

```python
def _make_enemy(self):
    margin = max(SCOUT_W, FIGHTER_W) // 2
    x = random.randint(margin, SCREEN_W - margin)
    speed_bonus = self.wave * WAVE_SPEED_FACTOR
    kamikaze_prob = min(WAVE_KAMIKAZE_CAP, WAVE_KAMIKAZE_BASE + self.wave * WAVE_KAMIKAZE_INC)
    if random.randint(1, 100) <= kamikaze_prob:
        k = Kamikaze(x)
        k.speed_bonus = speed_bonus
        return k
    fighter_prob = min(WAVE_FIGHTER_CAP, WAVE_FIGHTER_BASE + self.wave * WAVE_FIGHTER_INC)
    if random.randint(1, 100) <= fighter_prob:
        f = Fighter(x)
        f.speed_bonus = speed_bonus
        return f
    s = Scout(x)
    s.speed_bonus = speed_bonus
    return s
```

## Flujo completo

```
SpawnSystem._make_enemy():
  10-40% Kamikaze → target=player asignado en GameplayScene
  restante → Fighter o Scout (lógica existente)

Enemy muere:
  → score += points
  → spawn_system.register_kill()
  → 20% chance: PowerUp(random_kind) cae desde posición del enemy

PowerUp cae → player colisiona → player.apply_powerup(kind):
  extra_life   → lives = min(lives+1, 5)
  shield       → active_powerups.add("shield")
  rapid_fire   → active_powerups.add("rapid_fire")
  double_shot  → active_powerups.add("double_shot")
  triple_shot  → active_powerups.add("triple_shot")

player.shoot(now) → list[Bullet]:
  triple_shot  → 3 balas (centro, izq -16, der +16)
  double_shot  → 2 balas (izq -10, der +10)
  rapid_fire   → cooldown × 0.4
  normal       → 1 bala (centro)

player.take_damage(now):
  invincible   → return (ignorado)
  shield       → discard("shield"); return (sin perder vida)
  else         → lives -= 1; active_powerups.clear(); invincible 2s

render():
  shield activo → shield.png encima del player
  active_powerups → iconos 24×24 bottom-left
```

## Criterio de completitud v0.5

- Kamikaze aparece, persigue al player, usa sprite insect-1.png
- Enemy muerto dropea power-up con 20% de probabilidad
- Power-up cae hacia abajo, player lo recoge al tocarlo
- Shield absorbe 1 hit sin perder vida; desaparece al absorber
- RapidFire reduce cooldown de disparo a 40%
- DoubleShot dispara 2 balas por SPACE; TripleShot dispara 3
- ExtraLife: +1 vida (máx 5)
- Recibir daño sin shield: pierde vida + todos los power-ups se pierden
- Stack libre: RapidFire + DoubleShot simultáneos funcionan
- Iconos activos visibles bottom-left
- Shield visual (shield.png) sobre nave mientras activo
- Sin crashes en 5 minutos de juego
